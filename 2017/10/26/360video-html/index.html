HTML简介与文本

1.Doctype作用

  • 指定HTML页面使用的标准和版本。
  • 浏览器根据doctype决定使用哪种渲染模式。

2.渲染模式

  • 怪异模式（html4之前的版本）
  • 准标准模式（容错性较强）
  • 标准模式

3.XHTML

  • 用XML语法重新定义HTML
  • 语法要求严格。 之后XHTML2.0版本过于激进，问题多，不好用，以至于项目关闭。

4.HTML5设计思想

  • 兼容已有内容
  • 避免不必要的复杂性
  • 解决现实的问题
  • 优雅降级 （比如说在canvas里面嵌入html标签，当浏览器不支持的时候，会显示你嵌入的标签，对开发者友好，达到优雅降级）
  • 尊重事实标准
  • 用户>开发者>浏览器厂商>标准制定者>理论完美

5.HTML5中的变化。

  • 简化了doctype和meta标签。
  • 新增语义化标签和属性。(footer，header标签等等)
  • 去掉纯展示性标签(因为强调样式与结构分离，可以通过css实现)
  • canvas、video、audio、离线、本地存储、拖拽等等。

6.HTML5的语法。

  • 标签不区分大小写，推荐小写
  • 空标签可以不闭合，比如input、meta
  • 属性不必引号，推荐双引号
  • 某些属性值可以省略，比如required、readonly。 (它们的值为true or false，值为true的时候可以直接写属性而不用写值)

流式元素(flow)，此类元素通常包含文本或植入的内容,其实就是元素里面可以放入其他的标签。

7.HTML中的文本标签

  • HTML <hr> 元素表示段落级元素之间的主题转换（例如，一个故事中的场景的改变，或一个章节的主题的改变）。在HTML的早期版本中，它是一个水平线。现在它仍能在可视化浏览器中表现为水平线，但目前被定义为语义上的，而不是表现层面上。
  • <ol>标签 ，start可以设置顺序开始的数字。默认为1。
  • <ul>的<li>里面还可以嵌入<ul>。会自动缩进。
  • <dl>定义列表，用来表示一些键值对含义的列表。<dt>对应为键，<dd>对应为值。
  • 引用：<blockquote>对应为长段落的引用、<cite>用于比较短的一些名词的引用、<q>也可以表示引用，其与<blockquote>对应，它是表示一些短的语句的引用。

8.预格式化文本

  • <pre>标签可以保留标签内的文本的格式，比如说换行以及多个空格。
  • 展示代码一般用 <pre><code></code></pre>以顺利保留标签内文本格式。
  • <figure> 里面经常用于例如论文中的插图。

9.网页总体结构

10.强调

  • strong: 重要性、严重性和紧急性
  • em: 从一句话中突出某个词语
  • b: 将词语在视觉上和其他部分区分，比如论文摘要中的关键词
  • i: 换一种语调去说一句话，比如说其他语言翻译，旁白等。
  • <dfn>定义标签
  • <abbr>缩写标签，展示缩写的词
  • 上标<sup> 下标<sub>
  • <wbr>标签，正常情况下不显示，而在当包含其标签的文本需要换行的时候会在它出现的地方换行

11.实体(entity)字符

  • 空格 ‘&nbsp’;
  • < 小于号 ‘&lt’;
  • > 大于号 ‘&gt’; …………

链接与图片

1.<a href=””>查看图片</a>

  • href指定的url可以省略协议和host，省略之后协议及host与页面当前的协议保持一致。
  • chrome控制台中，$0可以打印出当前选中的html元素。

2.页面内锚点

  • href = “#” 会回到页面顶部，
  • href = “#haha” 页面会滚动到id 名为haha的元素位置。 通过页面内的锚点设置跳转到相关元素非常方便。

3.a标签中的target属性

可通过target属性控制链接打开的方式。默认是在当前窗口。

  • target=”_self”
  • target=”_blank”
  • target=”abc” 在同一个页面里面给target指定同一个值的链接都会在同一个target页面打开，这种设置可防止从一个页面打开过多页面。

4.<img src=”….” alt=”替代文字” width=”” height=””>

建议指定宽高，防止页面渲染的时候出现跳动。

指定图片宽高

  • 不指定宽高： 原图大小显示
  • 指定宽度：按比例缩放到指定宽度
  • 指定高度：按比例缩放发哦指定高度
  • 指定高宽：强制按指定宽高显示

5.常用图片格式

  • jpg 。 适合照片 颜色比较丰富，相对于png相同的像素占用空间小。
  • png 。色彩较少的时候使用，png24可以半透明。无损压缩，图片失真小。但是存储的索引色数量有限。
  • gif。无法半透明，可以做动画。
  • webp。谷歌新出的图片压缩格式。尺寸小，占用空间小，适合浏览器端使用。

HTML扩展知识

1.accesskey & tabindex

accesskey 全局属性提供了能为当前元素生成快捷键的方式。这个属性是由空格分隔的字符（单个 Unicode 代码点）列表组成。浏览器使用存在于计算机键盘布局的第一个字符。
accesskey 属性规定激活（使元素获得焦点）元素的快捷键。以下元素支持 accesskey 属性：, , ,


Windows Linux Mac
Firefox Alt + Shift + key On Firefox 14 or newer, Control + Alt + keyOn Firefox 13 or older, Control + key
Internet Explorer Alt + key N/A
Google Chrome Alt + key Control + Alt + key
Safari Alt + key N/A Control + Alt + key
Opera Shift + Esc opens a contents list which are accessible by accesskey, then, can choose an item by pressing key

tabindex 全局属性 是个整数，表示元素（如果可聚焦）是否能够接受输入焦点。 如果它应该参与键盘序列导航，那么就是它的位置。它可以设为多种值：

  • tabindex=负值 (通常是tabindex=“-1”)，表示元素是可聚焦的，但是不能通过键盘导航来访问到该元素，用JS做页面小组件内部键盘导航的时候非常有用。
  • tabindex=”0” ，表示元素是可聚焦的，并且可以通过键盘导航来聚焦到该元素，它的相对顺序是当前处于的DOM结构来决定的。
  • tabindex=正值，表示元素是可聚焦的，并且可以通过键盘导航来访问到该元素；它的相对顺序按照tabindex 的数值递增而滞后获焦。如果多个元素拥有相同的 tabindex，它们的相对顺序按照他们在当前DOM中的先后顺序决定。

根据键盘序列导航的顺序，值为 0 、非法值、或者没有 tabindex 值的元素应该放置在 tabindex 值为正值的元素后面。
